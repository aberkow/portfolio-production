Portfolio

Intro to CodePen

This was my introduction to both Thinkful and HTML/CSS. The goal was to create an "About me" page on CodePen based on a template. Since I had never done anything in HTML or CSS before, it was a big challenge for me. However I after getting help from my mentor, on Slack, and Google, I was able to complete it easily.

Languages: HTML and CSS

Karma Landing Page

This project demonstrates how I was able to expand my skills in order to make a copy of Karma's old landing page. I think my version is extremely close to the original. This challenged me to think about how a webpage is first wireframed and divided into different sections. From there, I worked on each section one at a time first organizing the HTML then applying the CSS. This project also helped me develop a better understanding of Chrome's dev tools and Git/Github. By the end of this project I understood much more about how websites can be organized, tested, and built.

Languages and skills: HTML, CSS, Chrome Dev Tools, Git/Github

Streetfighter!

In this project I learned how to implement jQuery to add functionality to a website. This project introduced basic programming logic to show and hide images. It also introduced CSS animation.

Languages: HTML, CSS, jQuery

Shopping List

This project was the most challenging so far. It wrapped together everything I knew about HTML, CSS, and jQuery. The first challenge was adding items to the list. After that, they needed the ability to be "checked off" and deleted. I completed the basic requirements and wanted to go a little further. After doing a little research I was able to sort the list items automatically and by dragging.

Languages: HTML, CSS, jQuery, jQueryUI

Hot or Cold

This challenge focused just on working through JavaScript functions. It also mimiced getting a pre-supplied HTML/CSS template from someone else and needing to add work to it. The issues involved included getting events to trigger at the right time, working out the logic of the game, and keeping track of all the data using a JavaScript object.

Languages: jQuery, JavaScript
